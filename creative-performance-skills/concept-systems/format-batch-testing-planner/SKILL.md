# Format Batch Testing Planner

**Category:** concept-systems
**Tier:** agency

## Purpose

This Skill plans a structured format-category batch test — a testing program where multiple concepts within the same creative format are deployed simultaneously so the format category itself is being evaluated, not just individual ads. Format batch testing answers the question "which format works for this audience and offer?" before spending resources optimizing within a format that may never work. It is the fastest way to find your category-level creative foundation.

---

## When to Use

- You're starting creative for a new brand or product and don't know which format performs best
- You want to test 3-4 format categories simultaneously and determine which one to invest in
- Your account has been running the same format for a long time and you want to test new categories
- You want to build a format-level performance map across multiple audience tiers
- You're planning a Q1/Q2 creative roadmap and want to structure it as a format discovery phase followed by an optimization phase

---

## Inputs Required

- Brand brief or product description
- Target audience(s)
- Available monthly test budget
- Number of format categories to test simultaneously (recommended: 3-4)
- Which specific format categories to include (or use recommended selection below)
- Testing timeline (4 weeks, 6 weeks, 8 weeks)
- Number of concepts per format category to test (recommended: 3-5 per category for valid signal)
- Campaign objective and primary metric

---

## What This Skill Does

Builds a complete format batch test plan: which formats to test first, how many concepts per format, budget allocation across formats, success criteria for declaring a format winner, sequencing for the optimization phase after format winners are identified, and a decision framework for what to do with results.

---

## Analysis Workflow

1. **Select the format categories for this test round**
   Recommended starting format batch for cold traffic:
   - Batch A (Trust-building): Founder Story, Testimonial, UGC Demo
   - Batch B (Logic-driven): Stat-Led Proof, Process Breakdown, FAQ Style
   - Batch C (Emotion-led): Before-After, Problem-Solution, Mistake-Driven
   - Batch D (Direct offer): Offer-Led Promo, Comparison, Objection Crusher

   Select 1-2 batches to run in Phase 1 based on audience awareness stage and product type.

2. **Set concepts per format category**
   Minimum 3 concepts per format to get format-level signal. If 1 of 3 concepts in a format performs well, the format may be valid but execution was inconsistent. If 0 of 3 perform well, the format is likely wrong for this audience.

3. **Allocate budget per format category**
   Equal spend allocation across format categories is required for valid comparison — do not allocate more to preferred formats. Minimum $150-300 per concept before reading results. Total budget estimate: (concepts per format) × (formats) × ($150-300 minimum per concept).

4. **Set format winner criteria**
   Define in advance: a format category wins if at minimum 2 of 3 concepts in that category outperform benchmark on the primary metric. A single outlier concept does not declare a category winner — consistency across the category does.

5. **Define the format verdict scale**
   - **Category Winner:** 3/3 or 2/3 concepts beat benchmark
   - **Category Potential:** 1/3 concepts beat benchmark — test more concepts in this format before deciding
   - **Category Loser:** 0/3 concepts beat benchmark at valid spend — deprioritize

6. **Plan the optimization phase (Phase 2)**
   After Phase 1 format testing: take the winning format category and expand the concept library within it. Only enter Phase 2 for confirmed Category Winners. Phase 2 tests variables within the winning format: hook variations, proof type variations, offer framing variations.

7. **Write the testing calendar**
   Week 1-2: Launch all format batch concepts (equal budget, no optimization)
   Week 3-4: Read results at the format category level, no individual ad decisions yet
   End of Phase 1: Declare format winners and losers, build Phase 2 plan

8. **Write the results decision framework**
   Exactly what decisions get made at the end of Phase 1:
   - Category Winner → Scale budget, brief 6-10 new concepts in this format
   - Category Potential → Brief 3 more concepts in this format for Phase 2 validation
   - Category Loser → Kill all concepts in this format, do not include in Phase 2

## Best Practices Applied

- Format-category testing is always Phase 1 — optimization is always Phase 2
- Equal budget across format categories is non-negotiable for valid format comparison
- Category consistency (2/3 concepts beating benchmark) is required to declare a format winner
- Phase 2 only begins for confirmed winners — optimizing a losing format category wastes resources
- The testing calendar prevents premature optimization decisions in the first two weeks

---

## Guardrails

- Do not allow more budget to flow to a preferred format during Phase 1 — this is the most common format test error
- Do not declare a format winner from a single performing concept — consistency across the category is required
- Do not run more format categories than the budget can support with statistically valid spend per concept
- Do not begin Phase 2 optimization before Phase 1 format verdicts are declared
- Do not run format batch tests and individual concept tests simultaneously in the same audience — they will interfere with each other
