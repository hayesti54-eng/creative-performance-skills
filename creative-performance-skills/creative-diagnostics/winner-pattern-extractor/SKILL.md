# Winner Pattern Extractor

**Category:** creative-diagnostics
**Tier:** pro

## Purpose

This Skill analyzes a set of top-performing creative assets to extract the repeatable patterns that are driving wins. It identifies what winning ads have in common across hooks, formats, proof elements, offer framing, visual style, and copy structure — then codifies those patterns into a replicable creative brief ruleset. The goal is to systematically scale what works instead of treating every winning ad as a lucky accident.

---

## When to Use

- You have 3+ winners in an account and want to understand what they have in common
- You're briefing new creative and want to anchor it in proven patterns from the account
- A client asks "why is this working?" and you need a rigorous answer
- You want to build a "Creative Playbook" for an account based on what has actually won
- You've scaled an ad and need to understand which element to protect when iterating
- You're preparing a new round of creative and want to increase your hit rate using past data

---

## Inputs Required

- 3-10 top-performing ad descriptions (copy, headline, visual description, or screenshot)
- Performance data for each winner (CTR, CVR, ROAS — the metric that defines "winning" for this account)
- Creative type / format for each winner
- Audience targeted for each winner
- Campaign objective
- Optional: hook rate and hold rate data for video winners

---

## What This Skill Does

Runs a cross-creative pattern analysis across your winners, identifying common elements in 7 dimensions: hook type, copy structure, proof element, offer framing, visual style, format, and awareness stage targeting. Outputs a pattern brief and a set of creative rules for this specific account.

---

## Analysis Workflow

1. **Catalog winner attributes**
   For each winning creative, extract and document: hook type, opening line verbatim, primary claim, proof element used, offer framing, CTA style, visual description, and format.

2. **Map hook patterns**
   Categorize the opening hook of each winner by type: outcome-first, problem-first, question, number-led, pattern interrupt, identity statement, social proof drop. Identify the most frequent hook type across winners.

3. **Map copy structure patterns**
   For each winner's body copy, identify whether it follows a recognizable DR structure and which moves appear in every winner: agitation, social proof, risk reversal, urgency, authority signal.

4. **Map proof patterns**
   What type of proof appears in winners? Specific numbers, testimonials, before-after, case study, stat-led, authority endorsement, certifications. Identify the proof format and specificity level that appears most consistently.

5. **Map offer framing patterns**
   How is the offer presented in each winner? Price anchoring, outcome-first, risk-reversal-first, bonus-led, urgency-led, simplicity-led. Document the dominant framing across the set.

6. **Map visual and format patterns**
   Common visual elements: text overlays, real vs. polished, face-forward vs. product-forward, lifestyle vs. studio, color usage, aspect ratio patterns. Common format patterns: video length, static style, carousel usage.

7. **Score pattern strength**
   For each identified pattern, score it: appeared in 3/3 winners = confirmed pattern; 2/3 = directional; 1/3 = coincidence. Only confirmed and directional patterns make it into the brief ruleset.

8. **Write the Winner Pattern Brief**
   Translate confirmed patterns into explicit creative rules for this account. Format as: "When creating ads for [brand/account], always [do X] because [confirmed pattern says X is present in Y% of winners]."

9. **Identify the one pattern to protect**
   Determine the single highest-confidence pattern — the one element that if removed would be most likely to break the creative. Label it as "the protected variable."

---

## Output Requirements

**Pattern Analysis Summary**
Table: Dimension | Pattern Found | Confidence Level | Appears In (X of Y winners)

**Winner Pattern Brief**
A set of creative rules derived from confirmed patterns. Formatted as directives: "Always...", "Never...", "Prioritize...", "Lead with..."

**The Protected Variable**
One clearly named element with explanation of why it must be preserved in all iterations of this creative.

**Replication Framework**
A brief that any copywriter or creative director can use to build the next creative with the highest probability of performing. Includes: hook type to use, copy structure to follow, proof to include, offer framing to lead with, visual direction.

---

## Best Practices Applied

- Pattern extraction requires at minimum 3 winners — below that, similarities are coincidence
- Patterns are extracted across dimensions, not just copy style
- Confidence scoring prevents treating one-off similarities as confirmed rules
- The "protected variable" concept prevents well-meaning iterations from accidentally removing what's working
- Winner patterns are account-specific, not universal — what works for one brand may not apply to another

---

## Guardrails

- Do not claim a pattern is confirmed with fewer than 3 data points
- Do not extract patterns from creatives with insufficient spend — data must be meaningful
- Do not conflate correlation with causation — patterns are directional signals, not guaranteed formulas
- Do not flatten nuance — if winners use different hooks, do not fabricate a fake common hook pattern
- Do not ignore format diversity — if winners span multiple formats, that itself is the pattern (format agnostic)
